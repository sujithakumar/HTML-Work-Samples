• The tabindex attribute specifies the tab order of an element (when the "tab" button is used for navigating).
  • Duplicate tabindex will be visited in order of its occurrence
  • tabindex=0 or no tabindex mentioned will be visited last, in the order of its occurrence
  • No tab index element will be visited before element with tabindex 0

TABINDEX    NULL/NO TABINDEX walmart

TABINDEX    2 Google

TABINDEX    1 Yahoo

TABINDEX    1 Amazon

TABINDEX    3 hotstar

TABINDEX    0 Netflix

Order
  1. Yahoo
  2. Amazon
  3. Google
  4. Hotstar
  5. walmart
  6. Netflix
